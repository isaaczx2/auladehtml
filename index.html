Aula 7 - Estudo de CSS

A sigla CSS é Cascading Style Sheets é uma linguagem utilizada para estilizar Página da web . Utilizando esta linguagem podemos definir:

  • Cor de fundo, texto e elemntos.
  • Fonte de tamanho das letras
  • Posicionamento de elemnetos dentro da página
  • Efeitos visuais e animações

O CSS funciona em conjunto com o HTML, enquanto o HTML diz o que parece, o CSS diz como deve aparecer.
